Paragraphs

Radiance uses the Roboto font family and the default font size is 16px. You can also add the leading class to any paragraph to highlight the first line (see above).

paragraph-example.html
<div class="container">
    <p class="leading">
        Radiance uses the Roboto font family and the default font size is 16px. You can also add the leading class to any paragraph to highlight the first line (see above).
    </p>
</div>

Line Breaks

Line breaks (or horizontal rules) are used to separate site sections, and sometimes text content. Line breaks are used by using a hr tag with the break class.

line-break-example.html
<div class="container">
    <h1>This is a Header</h1>
    <hr class="short center"/> <!-- Line Break! -->
    <p>This is the Text</p>
</div>

Block Quotes

Blockquotes are used to feature a specific quote used in an article, post, or anything really. Radiance proviced two types of block quotes, regular and fancy. Both types of blockquotes use the blockquote tag, and have styles set for child p tags, and a cite tag.

This is something that I said. It's a great quote. This should really be featured!

Chris Coppola

This is a featured quote or comment. This is for stuff that should be featured, but less highlighted than a fancy quote.

Chris Coppola

This is some non-featured text. This should wrap around any blockquotes with the left or right classes. This can be used to add variety, while still using the same style blockquotes. I am just rambling on now to make sure this example is visible.

blockquote-example.html
<div class="container">
	<blockquote class="fancy">
		<p>This is something that I said. It's a great quote. This should really be featured!</p>
		<cite>Chris Coppola</cite>
	</blockquote>
</div>

Colors

Radiance comes with a whole bunch of preset colors. You can also easily configure some Sass variables in order to customize the colors as you see fit. You can set the color of any element by adding the desired color class. (ex: red, shark, forest). You can also set text styles by using the color name, and adding a -text to the end. (ex: sky-text, violet-text, orange-text). For a full list of colors and how to customize them, see Colors

Grid System

Radiance has 2 types of grid systems. One uses boxes, and the other columns. Boxes auto-size based on how many are in a row or group by default. Columns are meant to be a set size, and sizes are set on a the column element.

Boxes

As stated above, boxes are meant to auto-resize. If you put 4 boxes in a row or group, they will each be 25% the width of their container. You can also set their sizes on the containing row or group. Sizes use breakpoint prefixes. See the below for examples of both. Note: Resize your browser.
