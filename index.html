Headers are used to separate text into different sections. All headers can be used by using a header tag (h1, h2, etc). Class tags are provided as well, to add header styles to any element (.h1, .h2, etc). 

<div class="container">
    <h1>Header 1</h1>
    <h2>Header 2</h2>
    <h3>Header 3</h3>
    <h4>Header 4</h4>
    <h5>Header 5</h5>
    <h6>Header 6</h6>
</div>
